## Getting Started

### Initial Setup

If you're setting up the school for the first time:

1. **Complete School Registration**
   - Navigate to school registration page
   - Enter school information
   - Set up school details

2. **Configure School Settings**
   - Go to **School Detail** (My School)
   - Enter complete school information
   - Configure academic calendar
   - Set up payment information

3. **Create Initial Users**
   - Add teachers
   - Add students
   - Assign roles appropriately

### Logging In

1. Navigate to **Admin Login** page
2. Enter your **username**
3. Enter your **password**
4. Click **Login**

## User Management

### Adding Users

#### Adding a Single User

1. Navigate to **Users** â†’ **Add User**
2. Fill in user information:
   - **Username** (for students, this is usually the index number)
   - **Email** (if available)
   - **Password** (or generate one)
   - **Role** (Student, Teacher, Admin, etc.)
   - **Additional Information** based on role
3. Click **Save**

#### Bulk User Creation

1. Go to **Users** â†’ **Bulk Add**
2. Prepare a list of users (Excel format recommended)
3. Upload the file or enter multiple users
4. Review the list
5. Click **Create Users**

### Managing Users

#### Editing User Information

1. Go to **Users**
2. Find the user you want to edit
3. Click **Edit**
4. Update information
5. Click **Save**

#### Activating/Deactivating Users

1. Navigate to **Users**
2. Find the user
3. Click **Status Change** or **Activate/Deactivate**
4. Confirm the action

#### Deleting Users

1. Go to **Users**
2. Find the user
3. Click **Delete**
4. Confirm deletion

> **Warning:** Deletion may be soft delete (user marked as deleted but data retained) or permanent. Check your school's policy.

### Managing Classes/Programs

#### Creating a Class

1. Navigate to **Classes**
2. Click **Add Class** or **Add Program**
3. Enter:
   - **Class Name** (e.g., "Form 1", "Grade 7")
   - **Class Code** (optional)
   - **Academic Year**
   - **Description** (optional)
4. Click **Save**

#### Editing Classes

1. Go to **Classes**
2. Find the class
3. Click **Edit**
4. Update information
5. Click **Save**

#### Deleting Classes

1. Navigate to **Classes**
2. Find the class
3. Click **Delete**
4. Confirm deletion

> **Note:** Ensure no students are enrolled before deleting a class.

#### Creating Multiple Classes

1. Go to **Classes** â†’ **Add Multiple**

## Results Management

### Viewing Results

1. Navigate to **Results**
2. Select **Academic Year** and **Term**
3. Choose **Class/Program**
4. View all results

### Approving Results

1. Go to **Results**
2. Find results with **Pending** status
3. Review the grades
4. Click **Approve** to publish or **Reject** to return

### Managing Result Approval

- **Approve All** - Approve all pending results for a class
- **Reject** - Return results for correction
- **Edit** - Make corrections if needed

### Result Workflow

Results follow this process:
1. **Teacher Enters Grades** â†’ Status: Pending
2. **Admin Reviews** â†’ Can approve or reject
3. **Approved** â†’ Published and visible to students
4. **Rejected** â†’ Returned to teacher for correction

## Troubleshooting

### Users Can't Log In

**Problem:** Users report login issues

**Solution:**
- Verify username is correct
- Reset password if needed
- Check if user account is active
- Verify user role assignments

### Grades Not Appearing

**Problem:** Entered grades not showing

**Solution:**
- Check if grades were saved
- Verify academic year/term selection
- Ensure subject is assigned to teacher
- Check result approval status

### Payment Issues

**Problem:** Payments not processing or recording

**Solution:**
- Verify Paystack configuration
- Check payment gateway settings
- Review payment logs
- Contact payment provider support if needed

### Export Not Working

**Problem:** Excel/PDF export fails

**Solution:**
- Check file permissions
- Verify data is available to export
- Try exporting smaller datasets
- Clear browser cache and try again
