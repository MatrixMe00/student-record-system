## First Time Setup

### Step 1: System Initialization

1. Navigate to `/setup` in your browser
2. Create the first **System Administrator** account
3. Enter administrator details:
   - Username
   - Email
   - Password
4. Click **Activate System**

> **Important:** This step can only be done once. Keep your administrator credentials safe.

### Step 2: School Registration

1. After system activation, you'll be redirected to school registration
2. Enter your school information:
   - School Name
   - Address
   - Contact Information
   - School Type
3. Complete the registration form
4. Submit to create your school

### Step 3: Login

1. Use your administrator credentials to log in
2. You'll be taken to the Dashboard
3. Complete your profile if prompted

## Initial Configuration

### Configure School Details

1. Click on your **Profile** (top-right)
2. Select **School Detail** or **My School**
3. Complete all school information:
   - Full school name
   - Physical address
   - Phone numbers
   - Email address
   - School logo (if enabled)
4. Click **Save**

### Set Up Payment Information

1. Navigate to **Payment Accounts**
2. Click **Add Payment Account**
3. Enter your payment details:
   - Bank account information
   - Payment gateway settings (Paystack)
4. Configure payment prices:
   - Result viewing fees
   - Other service fees
5. Save configuration

### Configure School Settings

1. Go to **School Settings** (if available)
2. Set up:
   - Academic calendar
   - Grading system
   - Result publication settings
3. Save settings

## Creating Your First Users

### Adding Teachers

1. Navigate to **Users**
2. Click **Add User** or **Bulk Add**
3. For each teacher, enter:
   - Username
   - Email (optional)
   - Password
   - Role: **Teacher**
   - Full name
   - Contact information
4. Click **Save**

### Adding Students

1. Go to **Users**
2. Click **Add User**
3. For each student, enter:
   - Username (will be index number)
   - Email (optional)
   - Password
   - Role: **Student**
   - Full name
   - Class/Program assignment
   - Contact information
4. Click **Save**

> **Tip:** Use **Bulk Add** to add multiple students at once using Excel import.

## Setting Up Academic Structure

### Step 1: Create Classes/Programs

1. Navigate to **Classes**
2. Click **Add Class**
3. Enter:
   - Class Name (e.g., "Form 1", "Grade 7")
   - Class Code (optional)
   - Academic Year
4. Click **Save**
5. Repeat for all classes

> **Tip:** Use **Add Multiple** to create several classes at once.

### Step 2: Create Subjects

1. Go to **Subjects**
2. Click **Add Subject**
3. Enter:
   - Subject Name (e.g., "Mathematics", "English")
   - Subject Code (optional)
4. Click **Save**
5. Repeat for all subjects

### Step 3: Assign Subjects to Teachers

1. Navigate to **Subjects** â†’ **Assign Teacher**
2. Select:
   - Subject
   - Teacher
   - Class (if applicable)
3. Click **Assign**
4. Repeat for all subject assignments

### Step 4: Assign Students to Classes

When adding students:
- Select their **Class/Program** during registration
- Or edit student profiles to change class assignment

## Common First Tasks

### Entering First Grades

1. Teachers log in
2. Navigate to **Results**
3. Select academic year, term, class, and subject
4. Enter grades for students
5. Submit for approval
6. Admin approves results
7. Students can view results

### Setting Up Payments

1. Configure payment gateway (Paystack)
2. Set payment prices
3. Create student bills (if needed)
4. Students can make payments
5. Payment status tracked automatically
